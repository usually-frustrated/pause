# Modern Typst Template

A contemporary resume template with accent colors and clean typography, built with Typst.

## Preview

Modern, colorful layout with blue accent colors and professional spacing.

## Features

- Blue accent color scheme
- Clean section headers with underlines
- Optimized spacing and typography
- Fast compilation with Typst
- Professional yet modern appearance

## Supported Resume Sections

- ✅ Basics (name, contact, location)
- ✅ Professional Summary
- ✅ Work Experience
- ✅ Education
- ✅ Skills
- ✅ Projects
- ✅ Awards & Honors

## Usage

```yaml
templates: |
  builtin:typst-modern
```

Or as shorthand:
```yaml
templates: typst-modern
```

## Output

Single-page PDF resume compiled with Typst CLI.
